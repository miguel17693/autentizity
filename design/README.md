# 📁 Carpeta de Diseño

Sube aquí todos los recursos visuales del proyecto:

## 📂 Estructura

- **`references/`** — Screenshots de webs de referencia, inspiración visual
- **`assets/`** — Logos, iconos, imágenes proporcionadas por el cliente
- **`notes.md`** — Notas de diseño y decisiones visuales

## 🎯 ¿Qué subir?

- Logo del proyecto/organización
- Paleta de colores (si existe)
- Tipografías deseadas
- Fotos o imágenes para el contenido
- Cualquier mockup o boceto
- Screenshots de referencias que te gusten

La IA leerá estos archivos para informar las decisiones de diseño.
