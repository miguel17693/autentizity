# 🗑️ Inbox — Carpeta temporal de input

Esta carpeta es para **tirar aquí cualquier cosa**: imágenes, capturas, textos, ideas, PDFs, lo que sea.

## Cómo funciona

1. **Tú metes archivos aquí** (imágenes, notas, lo que sea)
2. **Yo lo proceso** → extraigo la info útil y la muevo a donde toque:
   - Decisiones de diseño → `design/notes.md`
   - Assets útiles → `design/assets/`
   - Referencias → `design/references/`
   - Info de contenido → `AI_CONTEXT.md`
3. **Se borra de aquí** una vez procesado

> ⚠️ Todo lo que esté en esta carpeta es temporal. No se commitea nada importante aquí.
